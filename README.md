# quadratic-function
高校数学（数学I）の難所である「2次関数の最大・最小（場合分け）」を、直感的にマスターするための学習支援Webアプリケーションです。
ランダムに生成される問題（一般形）から自分で平方完成を考え、変数 $a$ をスライダーで動かすことで、境界値でどのように最大値・最小値が切り替わるのかをグラフのアニメーションと連動して確かめることができます。
---
## 主な機能
1. **4つの頻出パターンに対応**
   - **固定**: 定義域も軸もすべて具体的な数値。
   - **軸移動**: 軸が文字 $a$ で動くパターン（定義域固定）。
   - **範囲移動**: 定義域の片端が $a$ で動くパターン（軸固定）。
   - **幅移動**: 定義域の幅が一定のまま、$a$ の値に連動してスライドするパターン（軸固定）。
2. **上に凸・下に凸の両方に対応**
   - $x^2$ の係数が正（下に凸）だけでなく、負（上に凸）の問題もランダムで生成されます。
3. **一般形（展開された形）での出題**
   - 実戦的な演習ができるよう、式は展開された一般形で出題されます。
4. **動くグラフビジュアライザ**
   - 変数 $a$ のスライダーを動かすと、定義域や放物線がリアルタイムに動き、最大点(Max)と最小点(Min)の動きがひと目でわかります。
